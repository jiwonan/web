통신에 필요한 규약 : protocal(html은 http)
최초의 브라우저 : Netscape Nevigator
HTML5의 특징
  1. 반응형 웹
    모든 기기마다 문서가 동일한 구조로 보인다.
  2. 분리
    • 내용 : HTML5
    • 디자인 : CSS3
    • 동적인 기능 : javascript
  3. 구조
    <!DOCTYPE html>
    <html>
     <head>
      <title>제목이 될 내용</title>
     </head>
     <body>
      내용이 될 문서
     </body>
    </html>
  4. 태그가 쌍을 이룬다.(여는 태그가 있으면 닫는 태그가 있음.)
  5. 닫는 태그가 없는 것도 있다.(br, img, 등...)
